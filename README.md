# PY2-MICRO
Sistema de Carro Inteligente IoT con ESP8266 que integra control de acceso por RFID, encendido táctil y gestión térmica mediante BMP280 para activar un ventilador. Incluye faros automáticos según luz ambiental, asistencia de retroceso por proximidad y registro continuo en la nube para análisis avanzado con CUDA C++
